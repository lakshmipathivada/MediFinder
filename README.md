# MediFinder

MediFinder is an AI-powered medicine search application that helps users search medicines, compare prices, check availability, and find alternative medicines.

## Features
- Medicine Search
- Price Comparison
- Medicine Availability
- Alternative Medicines
- Interactive Map
- AI-powered Recommendations

## Technologies Used
- React
- TypeScript
- Vite
- Firebase
- Google AI Studio
- Gemini API

## Deployment
Google Cloud Run
